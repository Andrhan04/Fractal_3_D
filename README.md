# Fractal_3_D - Симуляция движения частиц в 3D фрактальной структуре

Проект для моделирования движения частиц в сложной 3D-геометрии, состоящей из последовательности наклонных цилиндров. Система включает генерацию частиц, симуляцию их движения, визуализацию и статистический анализ.

## 🎯 Основные возможности

- **Генерация частиц** в буферной зоне с различными начальными условиями
- **Симуляция движения** частиц в сложной фрактальной геометрии
- **Визуализация** 3D структур, позиций частиц и статистических распределений
- **Статистический анализ** времени жизни частиц и их пространственного распределения
- **Создание анимаций** и тепловых карт для анализа результатов

## 🏗️ Архитектура проекта

### Основные компоненты

#### 1. Ядро симуляции (C++)
- **Templates/** - Базовые классы:
  - `Point.h` - 3D точки и векторные операции
  - `Particle.h` - Частицы с движением и проверкой столкновений
  - `Fractal.h` - Фрактальная структура из цилиндров
  - `Bufer.h` - Буферная зона для генерации частиц
  - `Trap.h` - Ловушки для захвата частиц
  - `MyReader.h`/`MyWriter.h` - Чтение/запись конфигураций

#### 2. Генерация данных
- **Start_particle/** - Генерация начальных позиций частиц
- **Start_trap/** - Создание конфигураций ловушек
- **Sowing/** - Входные данные и конфигурации:
  - `Bufer/` - Параметры буферных зон
  - `Config/` - JSON конфигурации экспериментов
  - `Fractal/` - Геометрия фрактальных структур
  - `Particle/` - Начальные позиции частиц
  - `Trap/` - Конфигурации ловушек

#### 3. Симуляция
- **Move/** - Основная симуляция движения частиц
- **Move_with_traps/** - Симуляция с учетом ловушек

#### 4. Визуализация и анализ (Python)
- **Draw_figure/** - 3D визуализация структур и частиц
- **Statistics/** - Статистический анализ и построение графиков
- **images/** - Результирующие изображения и анимации

## 📊 Статистические возможности

### Анализ времени жизни
- **imperial_depends_from_traps.py** - Зависимость времени жизни от количества ловушек
- Логарифмическое масштабирование осей (1:10^6 соотношение X:Y)
- Построение графиков среднего времени жизни и количества выживших частиц

### Пространственный анализ
- **Histogramm.py** - Гистограммы распределения частиц по координатам
- **Heap_map.py** - Тепловые карты плотности частиц
- **Get_proections.py** - Проекции на координатные плоскости

### Анимации
- **Get_gif_histogram.py** - Анимированные гистограммы
- **Get_gif_proections.py** - Анимированные проекции

## 🚀 Быстрый старт

### Требования
- **C++17** компилятор с поддержкой OpenMP (MSVC/Visual Studio)
- **Python 3.10+** с пакетами:
  ```bash
  pip install numpy matplotlib pillow imageio
  ```

### Базовый workflow

1. **Подготовка конфигураций**
   ```bash
   # Генерация частиц
   cd Start_particle
   # Настройка config.txt и компиляция
   
   # Генерация ловушек (опционально)
   cd ../Start_trap
   # Настройка и компиляция
   ```

2. **Запуск симуляции**
   ```bash
   # Основная симуляция
   cd ../Move
   # Настройка input.json и компиляция
   
   # Симуляция с ловушками
   cd ../Move_with_traps
   # Настройка и компиляция
   ```

3. **Визуализация результатов**
   ```bash
   # 3D визуализация
   cd ../Draw_figure
   python Draw_trap_positions.py
   python Draw_point_positions.py
   
   # Статистический анализ
   cd ../Statistics
   python Histogramm.py
   python Heap_map.py
   python imperial_depends_from_traps.py
   ```

## 📁 Структура данных

### Входные форматы
- **JSON конфигурации** - `{"Bufer": id, "Fractal": id, "Size": value}`
- **Текстовые файлы** геометрии - координаты центров цилиндров
- **Текстовые файлы** частиц - начальные позиции (x y z)

### Выходные данные
- **Experiments/exp_{id}.txt** - Результаты симуляций
- **Results/experiment_{exp_id}.json** - Метаданные экспериментов
- **images/Experiment_{id}/** - Визуализации и графики

## 🎨 Визуализация

### 3D рендеринг
- Сферы для ловушек с заданными радиусами
- Цилиндрические сегменты фрактальной структуры
- Трехмерные траектории частиц
- Прозрачные материалы для лучшего обзора

### Статистические графики
- Гистограммы с автоматическим расчетом интервалов
- Тепловые карты плотности распределения
- Графики зависимостей с логарифмическими шкалами
- Русскоязычные подписи и метки

## 🔧 Технические особенности

### Оптимизации
- Эффективное удаление "мертвых" частиц через swap алгоритм
- Поддержка многопоточности через OpenMP
- Оптимизированные структуры данных для быстрого доступа

### Особенности реализации
- Пользовательская функция swap для класса Particle
- Поддержка наклонных цилиндров с основаниями перпендикулярными OX
- Гибкая система конфигураций через JSON
- Расширенная система визуализации на matplotlib
