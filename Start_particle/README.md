## Start_particle

Генерирует стартовые точки частиц в буферной зоне и формирует конфигурационный JSON для симуляции движения частиц в 3D фрактальных структурах.

### Алгоритм генерации частиц

**Процесс генерации:**
1. Чтение геометрии буфера из `Sowing/Bufer/Bufer_{id}.txt`
2. Создание виртуального наклонного цилиндра по заданным точкам
3. Генерация `N` равномерно распределенных точек внутри цилиндра
4. Проверка принадлежности каждой точки объему цилиндра
5. Сохранение валидных точек и формирование конфигурации

**Математическая основа:**
- Используется равномерное распределение в цилиндрических координатах
- Преобразование координат с учетом наклона цилиндра
- Эффективные алгоритмы проверки принадлежности точки цилиндру
- Минимизация вычислений через аналитические решения

### Входные данные

**Файл `config.txt`:**
```
BUFER_ID FRACTAL_ID N SIZE
```

**Параметры:**
- `BUFER_ID` — ID буферной зоны (соответствует `Sowing/Bufer/Bufer_{id}.txt`)
- `FRACTAL_ID` — ID фрактальной структуры (соответствует `Sowing/Fractal/Pole_{id}.txt`)
- `N` — количество генерируемых частиц (целое положительное число)
- `SIZE` — радиус цилиндров для буфера и фрактала (вещественное число > 0)

**Требования к входным данным:**
- Файл буфера должен существовать и содержать ровно 2 точки
- Файл фрактала должен существовать и содержать хотя бы 2 точки
- Параметры должны быть корректными числами

### Выходные данные

**1. Файл с частицами:** `Sowing/Particle/points_{id}.txt`
```
x1 y1 z1
x2 y2 z2
...
xN yN zN
```

**Особенности:**
- Каждая строка содержит координаты одной частицы
- Координаты в декартовой системе
- Все точки гарантированно находятся внутри буферного цилиндра
- Порядок точек случайный (равномерное распределение)

**2. Конфигурационный файл:** `Sowing/Config/Conf_{id}.json`
```json
{
  "Bufer": BUFER_ID,
  "Fractal": FRACTAL_ID,
  "Point": AUTO_GENERATED_ID,
  "Size": SIZE
}
```

**Особенности:**
- Автоматическая генерация ID для точек
- Связывание всех компонентов симуляции
- Сохранение параметров для воспроизводимости

### Запуск программы

1. **Подготовка**: Убедитесь, что файлы буфера и фрактала существуют
2. **Создание config.txt**: Подготовьте файл с параметрами генерации
3. **Компиляция**: Соберите проект в Visual Studio (требуется C++17)
4. **Запуск**: Выполните программу из каталога с config.txt

**Пример запуска:**
```bash
# Перейдите в каталог с исполняемым файлом
cd Start_particle

# Запустите программу
Start_particle.exe
```

### Обработка ошибок

Программа включает comprehensive error handling:
- Проверка существования входных файлов
- Валидация формата и содержания файлов
- Контроль корректности параметров генерации
- Сообщения об ошибках с указанием конкретных проблем
- Graceful degradation при некритичных ошибках

### Производительность

**Оптимизации:**
- Эффективные алгоритмы генерации случайных точек

**Производительность:**

### Использование в симуляции

Сгенерированные данные используются:
1. Как входные данные для Move.cpp
2. Для визуализации в Draw_figure скриптах
3. Для анализа в Statistics скриптах
4. Для воспроизводимости экспериментов
