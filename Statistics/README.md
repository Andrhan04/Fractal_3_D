# Statistics - Скрипты для анализа статистики экспериментов

Эта папка содержит скрипты для анализа и визуализации статистических данных экспериментов с фракталами.

## Скрипты

### 1. [Histogramm.py](file:///c:/Users/user/source/Fractal_3_D/Statistics/Histogramm.py)
**Назначение**: Строит гистограммы распределения частиц по координатам X, Y, Z для всех шагов эксперимента.

**Функциональность:**
- Читает параметры эксперимента из файла `input.json`
- Извлекает данные о конфигурации, фрактале и буфере
- Собирает координаты частиц из всех запусков эксперимента
- Создает гистограммы для каждого шага эксперимента
- Сохраняет изображения гистограмм в папку `../images/Experiment_{id}/Hist/`
- Автоматически рассчитывает количество интервалов на основе размера категории
- Визуализирует распределение по всем трем координатам на одном графике

**Использование:**
```bash
python Histogramm.py
```

### 2. [Get_gif_histogram.py](file:///c:/Users/user/source/Fractal_3_D/Statistics/Get_gif_histogram.py)
**Назначение**: Создает анимированный GIF из последовательности гистограмм.

**Функциональность:**
- Читает ID эксперимента из файла `input.json`
- Находит все файлы гистограмм в папке `../images/Experiment_{id}/Hist/`
- Сортирует файлы по номеру шага
- Создает анимированный GIF с длительностью кадра 0.5 секунды
- Сохраняет GIF в папку `../images/Experiment_{id}/`

**Использование:**
```bash
python Get_gif_histogram.py
```

### 3. [input.exmple.json](file:///c:/Users/user/source/Fractal_3_D/Statistics/input.exmple.json)
**Назначение**: Пример файла конфигурации для скриптов.

**Содержит:**
- `exp_id` - ID эксперимента (число)
- `category_len` - размер категории/интервала для гистограмм

**Использование:**
Скопируйте файл в `input.json` и настройте параметры для вашего эксперимента.

## Порядок использования

1. Настройте `input.json` с параметрами эксперимента
2. Запустите `Histogramm.py` для создания гистограмм
3. Запустите `Get_gif_histogram.py` для создания анимации

## Зависимости

- `matplotlib` - для построения графиков
- `numpy` - для численных вычислений
- `PIL` (Pillow) - для работы с изображениями
- `imageio` - для создания GIF анимаций
- `json` - для работы с JSON файлами
