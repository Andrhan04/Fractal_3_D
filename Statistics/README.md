# Statistics - Скрипты для анализа статистики экспериментов

Эта папка содержит скрипты для анализа и визуализации статистических данных экспериментов с фракталами.

## 📊 Доступные скрипты

### 1. [Histogramm.py](file:///c:/Users/user/source/Fractal_3_D/Statistics/Histogramm.py)
**Назначение**: Строит гистограммы распределения частиц по координатам X, Y, Z для всех шагов эксперимента.

**Функциональность:**
- Читает параметры эксперимента из файла `input.json`
- Извлекает данные о конфигурации, фрактале и буфере
- Собирает координаты частиц из всех запусков эксперимента
- Создает гистограммы для каждого шага эксперимента
- Сохраняет изображения гистограмм в папку `../images/Experiment_{id}/Hist/`
- Автоматически рассчитывает количество интервалов на основе количества категорий
- Визуализирует распределение по всем трем координатам на одном графике
- Поддерживает русскоязычные подписи и метки

**Ключевые особенности:**
- **Переменная `num_categories`** - количество интервалов/категорий вместо размера шага
- **Автоматическое масштабирование** - адаптивные границы интервалов
- **Нормализация данных** - автоматическое приведение к удобным масштабам
- **Пакетная обработка** - обработка всех шагов эксперимента

**Использование:**
```bash
python Histogramm.py
```

### 2. [Get_gif_histogram.py](file:///c:/Users/user/source/Fractal_3_D/Statistics/Get_gif_histogram.py)
**Назначение**: Создает анимированный GIF из последовательности гистограмм.

**Функциональность:**
- Читает ID эксперимента из файла `input.json`
- Находит все файлы гистограмм в папке `../images/Experiment_{id}/Hist/`
- Сортирует файлы по номеру шага
- Создает анимированный GIF с длительностью кадра 0.5 секунды
- Сохраняет GIF в папку `../images/Experiment_{id}/`

**Использование:**
```bash
python Get_gif_histogram.py
```

### 3. [Get_proections.py](file:///c:/Users/user/source/Fractal_3_D/Statistics/Get_proections.py)
**Назначение**: Создает проекции частиц, фрактала и буфера на плоскости OXY, OXZ, OYZ.

**Функциональность:**
- Читает параметры эксперимента из файла `input.json`
- Извлекает данные о конфигурации, фрактале и буфере
- Обрабатывает данные из одного запуска эксперимента (без усреднения)
- Создает проекции на все три плоскости: OXY, OXZ, OYZ
- Визуализирует частицы, фрактал и буфер с цветовой схемой:
  - Частицы: зеленый
  - Фрактал: синий (прозрачный в 3D)
  - Буфер: красный (прозрачный в 3D)
- Использует геометрическое представление:
  - Для OXY/OXZ: прямоугольники между точками с размером 2*rad
  - Для OYZ: окружности радиусом rad, соединенные прямоугольниками
- Сохраняет изображения проекций в папку `../images/Experiment_{id}/Projections/`
- Создает как 2D проекции, так и 3D визуализацию

**Использование:**
```bash
python Get_proections.py
```

### 5. [Heap_map.py](file:///c:/Users/user/source/Fractal_3_D/Statistics/Heap_map.py)
**Назначение**: Создает тепловые карты распределения плотности частиц.

**Функциональность:**
- Читает данные частиц для указанного шага эксперимента
- Создает 2D гистограммы плотности для плоскостей XY, XZ, YZ
- Генерирует тепловые карты с цветовой градацией
- Сохраняет изображения в нескольких форматах:
  - `../images/Experiment_{id}/Heap_map/step_{step}/heatmap.png`
  - `../images/Experiment_{id}/Heap_map.png` (финальная карта)
- Автоматически создает необходимые директории
- Обрабатывает все шаги эксперимента с заданным интервалом

**Использование:**
```bash
python Heap_map.py
```

### 6. [imperial_depends_from_traps.py](file:///c:/Users/user/source/Fractal_3_D/Statistics/imperial_depends_from_traps.py)
**Назначение**: Анализ зависимости времени жизни частиц от количества ловушек.

**Функциональность:**
- Анализирует результаты экспериментов с различным количеством ловушек
- Строит графики зависимости среднего времени жизни от количества ловушек
- Создает графики количества выживших частиц
- **Логарифмическое масштабирование** с соотношением осей 1:10^6
- Автоматическая обработка данных из multiple экспериментов
- Сохраняет графики в высоком разрешении:
  - `../image_depends/imperial/experiment_{id}_time_live.png`
  - `../image_depends/imperial/experiment_{id}_alive_cnt.png`

**Ключевые особенности:**
- **Точное масштабирование** - соотношение 1 единица по X : 10^6 единиц по Y
- **Оптимизированные метки** - читаемые подписи на логарифмической шкале
- **Статистическая обработка** - усреднение по повторным запускам
- **Профессиональное оформление** - сетка, легенды, подписи осей

**Использование:**
```bash
python imperial_depends_from_traps.py
```

## 📋 Порядок использования

1. **Настройте конфигурацию** - Скопируйте `input.exmple.json` в `input.json`
2. **Запустите анализ** - Выполните нужные скрипты в последовательности:
   ```bash
   # Базовый анализ
   python Histogramm.py
   python Get_gif_histogram.py
   
   # Пространственный анализ  
   python Get_proections.py
   python Get_gif_proections.py
   
   # Плотностный анализ
   python Heap_map.py
   
   # Статистический анализ
   python imperial_depends_from_traps.py
   ```

3. **Просмотрите результаты** - Все изображения сохраняются в `../images/Experiment_{id}/`

## ⚙️ Настройка параметров

### Для гистограмм (Histogramm.py)
- `num_categories` - количество интервалов разбиения (вместо старого `category_len`)
- Автоматический расчет границ интервалов на основе диапазона данных

### Для тепловых карт (Heap_map.py)
- Автоматическое определение шага обработки (по умолчанию каждые 100,000 шагов)
- Настраиваемое разрешение тепловых карт

### Для имперского анализа (imperial_depends_from_traps.py)
- Фиксированное соотношение масштабов осей 1:10^6
- Логарифмическая шкала с шагом 10^5 по оси Y
- Автоматическое определение границ данных
