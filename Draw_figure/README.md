## Draw_figure

Скрипты визуализации буфера, фрактала и позиций частиц в 3D пространстве.

### Зависимости
- Python 3.10+
- `pip install numpy matplotlib pillow`

### Скрипты

#### 1. [paint_fractal.py](file:///c:/Users/user/source/Fractal_3_D/Draw_figure/paint_fractal.py)
**Назначение**: Визуализация геометрии буфера и фрактала с заданным радиусом.

**Функциональность:**
- Читает геометрию из файлов:
  - `Sowing/Bufer/Bufer_{id}.txt` — один наклонный цилиндр (две точки)
  - `Sowing/Fractal/Pole_{id}.txt` — последовательность центров наклонных цилиндров
- Создает 3D визуализацию с использованием `Poly3DCollection`
- Рисует цилиндрические сегменты с основаниями перпендикулярными оси OX
- Поддерживает наклонные цилиндры и сложные фрактальные структуры
- Сохраняет изображения в высоком разрешении

#### 2. [Draw_point_positions.py](file:///c:/Users/user/source/Fractal_3_D/Draw_figure/Draw_point_positions.py)
**Назначение**: Визуализация результатов эксперимента вместе с геометрией и стартовыми точками.

**Функциональность:**
- Читает геометрию буфера и фрактала
- Загружает результаты эксперимента из `Experiments/exp_{id}.txt`
- Использует метаданные из `Results/experiment_{exp_id}.json`
- Отображает позиции частиц в 3D пространстве вместе с геометрией
- Поддерживает интерактивное вращение и масштабирование 3D сцены
- Сохраняет комплексные визуализации экспериментов

#### 3. [draw_fbx.py](file:///c:/Users/user/source/Fractal_3_D/Draw_figure/draw_fbx.py)
**Назначение**: Экспорт 3D геометрии в форматы FBX/OBJ для использования в других приложениях.

### Настройки
- `draw_point.json` (для `Draw_point_positions.py`):
  - `exp_id` — ID результата в каталоге `Results/experiment_{exp_id}.json`
  - `try_id` — индекс запуска внутри файла результатов

### Запуск
1. Убедитесь, что каталоги `Sowing` и `Results/Experiments` заполнены результатами симуляций.
2. Запустите скрипт:
   
   ```bash
   python paint_fractal.py
   ```
   
   или
   
   ```bash
   python Draw_point_positions.py
   ```

### Особенности 3D визуализации
- **Интерактивное управление**: вращение, масштабирование, панорамирование
- **Качественное отображение**: сглаживание, тени, прозрачность
- **Экспорт**: поддержка форматов PNG, SVG, FBX, OBJ
- **Автоматическое расположение**: оптимальное размещение камеры и освещения

### Что делает программа
- Читает геометрию из текстовых файлов и преобразует в 3D модели
- Создает точные цилиндрические представления фрактальных структур
- Визуализирует движение частиц в контексте 3D геометрии
- Предоставляет инструменты для анализа пространственного распределения
- Сохраняет результаты в форматах, пригодных для публикации и дальнейшего анализа
