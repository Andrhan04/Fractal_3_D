# Sowing - Данные для симуляции

Эта папка содержит входные данные для симуляции движения частиц.

## Структура

### Config
Содержит конфигурационные JSON файлы, которые связывают:
- `Bufer` - ID буферной зоны
- `Fractal` - ID фрактальной структуры  
- `Point` - ID файла с частицами
- `Size` - радиус цилиндров

### Bufer
Содержит файлы с центрами буферных цилиндров:
- Каждый файл `Bufer_{id}.txt` содержит две строки с координатами центров
- Буфер представляет собой наклонный цилиндр

### Fractal  
Содержит файлы с центрами фрактальных цилиндров:
- Каждый файл `Pole_{id}.txt` содержит последовательность центров (x y z на строку)
- Фрактал состоит из последовательности наклонных цилиндров

### Particle
Содержит стартовые позиции частиц:
- Файлы `points_{id}.txt` с координатами частиц
- Частицы генерируются внутри буферной зоны
