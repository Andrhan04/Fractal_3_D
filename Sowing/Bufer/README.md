# Bufer - Буферные зоны

Файлы содержат центры буферных цилиндров для генерации стартовых позиций частиц.

## Формат файлов

Каждый файл `Bufer_{id}.txt` содержит две строки с координатами центров:
- Первая строка: координаты нижнего центра цилиндра
- Вторая строка: координаты верхнего центра цилиндра  

## Структура данных

- Буфер представляет собой наклонный цилиндр
- Ось цилиндра проходит через два заданных центра
- Радиус цилиндра задается в конфигурационном файле
- Частицы генерируются равномерно внутри этого цилиндра

## Пример файла Bufer_0.txt
```
0.0 0.0 0.0
10.0 5.0 3.0
```
